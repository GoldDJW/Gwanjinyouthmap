광진구 청소년자원지도 - 카카오 지도 전환 패키지

1. 목적
- 기존 '지도제작용_최종' 시트의 기관 목록과 분류를 유지합니다.
- 지도 엔진만 카카오 지도 JavaScript API로 변경합니다.
- 공개용 화면에서는 이용자가 기관 파일을 업로드하지 않습니다.
- 동일 건물은 하나의 마커에 기관목록으로 표시합니다.

2. 가장 쉬운 준비 방법
1) package_builder.html을 Chrome 또는 Edge로 엽니다.
2) 원본 엑셀 '유관기관현황자료_정확좌표_분포분석_레이어설계완료.xlsx'를 선택합니다.
3) 카카오 Developers에서 발급받은 JavaScript 키를 입력합니다.
4) '공개용 ZIP 만들기'를 누릅니다.
5) 내려받은 ZIP을 압축 해제하고 웹서버에 업로드합니다.

3. 카카오 설정
- 카카오 Developers 앱 생성
- 카카오 지도 API 활성화
- JavaScript 키 확인
- 실제 운영 도메인을 JavaScript SDK 도메인에 등록
- 로컬 시험 시에는 사용할 localhost 주소도 등록해야 합니다.

4. 파일 구조
index.html                 시작화면
youth_map.html             청소년용 서비스 찾기
practitioner_map.html      실무자용 관리·분석
config.js                  카카오 JavaScript 키와 기본설정
data/institutions.js       기관 데이터(자동 로딩)
data/gwangjin_boundary.js  선택적 로컬 행정동 경계
assets/                    공통 화면과 지도 기능

5. 서비스 필터
- 실무자용에서 서비스 미선택: 기관 미표시
- '전체' 선택: 모든 서비스 기관 표시
- 개별 서비스 선택: 선택 분야 중 하나 이상 제공하는 기관 표시

6. 데이터 갱신
- 원본 엑셀을 수정한 후 package_builder.html에서 새 ZIP을 만듭니다.
- 웹서버의 data/institutions.js만 교체해도 기관 데이터가 갱신됩니다.
- 일반 이용자는 기관 파일을 선택하거나 업로드하지 않습니다.

7. 유의사항
- 카카오 지도는 등록된 도메인과 올바른 JavaScript 키에서만 동작합니다.
- 행정동 경계는 공개 GeoJSON 자동 연결을 기본으로 사용합니다.
- 10분 생활권은 기본 반경 700m의 직선거리 근사값입니다.
- 외부연계기관은 실무자용 화면에서 별도로 켤 수 있습니다.


8. 기관 세부정보 기능
- 지도 마커의 기존 정보는 유지됩니다.
- 기존 홈페이지 링크는 세부정보 버튼으로 변경됩니다.
- 세부정보 페이지에서 기관소개, 사업소개, 홈페이지 주소 및 바로가기, 카카오맵 길찾기를 제공합니다.
- 원본 엑셀에 기관소개·사업소개 열이 포함되어야 합니다.
